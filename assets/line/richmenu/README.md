# リッチメニュー画像

## 配置するファイル

- `richmenu.png` または `richmenu.jpg` - リッチメニュー全体の画像

## 画像仕様

| 項目 | 値 |
|------|-----|
| サイズ | **2500×1686px**（大）または **2500×843px**（小） |
| 形式 | PNG または JPEG |
| ファイルサイズ | 1MB以下 |
